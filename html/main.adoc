== Introducción
Éste documento de texto define las reglas de estilo y formato para HTML.

Su objetivo es mejorar la colaboración y la calidad del código.

Ésto se aplica al código fuente, no al código que se genera usando herramientas (ofuscado, minificado o compilado).
